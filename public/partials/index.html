Copy and paste the code below into your terminal:
                            
TOOLSET_HOST="http://toolset.laurasjourney.nl"
TOOLSET_PROJECT_DIR_BASE="$HOME/reveal-js-pres"
TOOLSET_PROJECT_DIR="$TOOLSET_PROJECT_DIR_BASE"
TOOLSET_PROJECT_DIR_COUNTER=0
while [[ -d "$TOOLSET_PROJECT_DIR" ]] ; do
TOOLSET_PROJECT_DIR_COUNTER=`expr $TOOLSET_PROJECT_DIR_COUNTER + 1`
TOOLSET_PROJECT_DIR="$TOOLSET_PROJECT_DIR_BASE-$TOOLSET_PROJECT_DIR_COUNTER"
done
mkdir $TOOLSET_PROJECT_DIR && cd $TOOLSET_PROJECT_DIR

curl $TOOLSET_HOST/static/downloads/pres-base-0.1/pres.tar | tar xv

docker run -p 8000:8000 -p 35729:35729 -v $(pwd)/pres/:/revealjs/pres/:ro containersol/pres-base:0.1
Description

Now go to localhost:8080. The presentation resides in your project directory; why not open a new tab and edit pres/slides.md?

Note that the image run RevealJS 2.6 and ot yet 3.0. It has been tested in Chrome and been found to be a bit buggy in Firefox.
